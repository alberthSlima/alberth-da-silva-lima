# Portfólio Pessoal — Alberth Lima

Currículo online desenvolvido para a atividade prática de **Fundamentos da Programação Web**.

Site feito com **HTML5, CSS3 e JavaScript**.

## 📄 Páginas

| Página | Arquivo | Conteúdo |
|--------|---------|----------|
| Sobre mim | `index.html` | Apresentação pessoal e hobbies |
| Formação | `formacao.html` | Formação acadêmica, cursos e idiomas |
| Portfólio | `portfolio.html` | Projetos desenvolvidos |
| Contato | `contato.html` | Formulário de contatao |

## ⚙️ Funcionalidades em JavaScript

- Validação do formulário de contato (nome, e-mail e mensagem).
- Verificação do formato do e-mail.
- Simulação de envio com mensagem de confirmação (modal) e limpeza dos campos.
- Alternância entre tema claro e escuro (salvo no navegador).
- Menu responsivo (hambúrguer) para celulares.
